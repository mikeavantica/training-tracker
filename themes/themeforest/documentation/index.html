Markup Markup customization and highlights

× Important Note! Make sure to start with the template.html , it consists of basic markup of the template

This theme is a responsive layout with bootstrap 3.0 The theme consists of primarily two columns nested in class .frame All of the information within the main content area is nested within a div with an class of .content The sidebar's (column #2) content is within a div with an class of .sidebar

Once we are into the content div , we can use twitter bootstrap panels nested with in a class row. For more info on rows, columns and panels visit twitter bootstrap documentation. Lets dig up this more deep to have real fun

Styles Styles customization and highlights

Heads up! I'm using two CSS files in this theme.

Stylesheets

Here's a list of the stylesheet files I'm using with this template, you can find more information opening each file:

File Name Description
archon.css This is the main css file of the theme. Other style files are included at the top of this file using @import.
responsive.css This file consists of all override classes of the main classes when on other devices. It is complete set of media queries (@media)
custom.css Create a file with the name custom.css and include it below the responsive.css , in this file you can write any custom classes to override the template.
